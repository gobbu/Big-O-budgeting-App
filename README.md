Big O Budgeting App
Created by: Andy Lee, Guo Wei Xu, Wesley Meah, Ryan Cao, Louis Zeng. 
Public copy of a project originally developed in a private classroom repository.

Overview

The Big O Budgeting App is a web application that helps users manage their personal finances. Users can track expenses, create budgets for different categories, and visualize spending trends with interactive charts.

Features

User Accounts: Signup, login, and personalized dashboards

Budget Management: Create, edit, and delete budget categories

Expense Tracking: Add and monitor spending across categories

Data Visualization: Pie charts and summaries for monthly budget vs. spent amounts

Responsive Design: Works on desktop and mobile devices
